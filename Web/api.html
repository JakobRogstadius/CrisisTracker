API endpoints and accepted parameters

RSS feed (Atom 1.0)
No parameters.
get_stories.php
ongoing 0 or 1 Rank stories by how much they have ever been discussed (0), or by how much they have been discussed in the past four hours (1).
mintime ISO time, e.g. 2013-07-02 17:01:14 Earliest timestamp of the first tweet in a story.
maxtime ISO time, e.g. 2013-07-02 17:01:14 Latest timestamp of the first tweet in a story.
text E.g. damascus,children Comma separated keyword filter for stories. Multiple keywords are combined with AND.
topics E.g. 1:141,815;256:149 Semi-colon separated list of attributeID:labelID,labelID sets, with IDs matching the AIDR instance. Multiple label filters are combined with OR inside the same attribute, and AND between attributes.
get_story_info.php
ID (required) E.g. 20294793 A story ID for which to return basic story meta-data.
get_story_tweets.php
ID (required) E.g. 20294793 A story ID for which to return a sample of tweets contained in the story.
mintime ISO time, e.g. 2013-07-02 17:01:14 Earliest timestamp of the first tweet returned.
maxtime ISO time, e.g. 2013-07-02 17:01:14 Latest timestamp of the last tweet returned.
text E.g. video Comma separated keyword filter for tweets. Multiple keywords are combined with AND.
label_id E.g. 149 A label ID filter for tweets in the story, matching an ID from the AIDR instance.
get_aidr_topic_filters.php
Returns available topic filters from the AIDR instance.
