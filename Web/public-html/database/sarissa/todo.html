Overview

This document contains features users would like to see in Sarissa. To add your own post them to the forum or mailing list and we will add it here.

AJAXSLT Integration

A wrapper implementing XSLTProcessor around Google's AJAXSLT as a fallback for browsers without native scripting hooks for XSLT.

JSP Taglibs

JSP Tag Libraries would make Sarissa easier to use for server side developers without requiring scripting knowledge. A typical set of tag libraries would include AJAX forms, tables and polling widgets.

Struts 2 Integration

Similar to the JSP Tag Libraries above, but tied to the Struts 2 web framework.

JSF Integration

Similar to the JSP Tag Libraries above, based on JavaServer Faces .

Ruby Integration

Similar to the JSP Tag Libraries concept above, but designed for Ruby Frameworks, such as Ruby on Rails .

Python Integration

Similar to the JSP Tag Libraries concept above, but designed for Python Web Frameworks .
