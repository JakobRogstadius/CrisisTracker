Sarissa Ajax Form

To submit a form using AJAX with Sarissa, all you need to do is add a call to Sarissa.updateContentFromForm in the form's onsubmit attribute:

<form action="test.xml" method="post"
    onsubmit="return Sarissa.updateContentFromForm(this, document.getElementById('container'));">

Where 'container' is the id of the target element. In our case, this is the black-bordered div bellow. To change it's content, just submit the form using the submit button.
